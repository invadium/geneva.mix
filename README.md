# geneva.mix

A test set for Collider.JAM framework

## How to Run

Run collider in the main directory:

```
jam -d
```

Or you could use the ```probe``` script to start directly in the default browser:

```
./probe
```

To run silently, use ```--silent``` flag:

```
jam -d --silent
```

Or, alternatively:

```
./probe --silent
```




## Testing with Bootstrap

```
./test
```
